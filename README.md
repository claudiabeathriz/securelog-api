# SecureLog API

Backend API with focus on security, authentication and audit logging inspired by real corporate systems.

## Technical Stack
- Java
- Spring Boot
- PostgreSQL
- JPA / Hibernate

## Features
- Criar usuário
- Listar usuários
- Buscar usuário por ID
- Hash de senha com BCrypt
- Validação de entrada

## Como rodar
1. Clonar repo
2. Configurar PostgreSQL
3. Rodar aplicação

## Endpoints
POST /users
GET /users
GET /users/{id}
